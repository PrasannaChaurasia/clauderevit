# ClaudeRevit — Roadmap

## Completed (v1.0)

- [x] Claude Panel: 10 buttons (Start/Stop Listener, Claude Command, Build Model, Generate Views, Place Rooms, Make Schedule, Create Sheets, Model Audit, Ask Claude)
- [x] Script Panel: Python Script editor + C# Script editor
- [x] Automate Panel: Auto-Dimension, Auto-Tag, Auto-Annotate, Sheet Template
- [x] Audit Panel: Arch Audit, MEP Audit, Structural Audit (UK/EU/US/International)
- [x] Parametric Panel: Parametric Model, DirectShape, Family Browser
- [x] MCP Server: 14 tools, Claude Desktop integration
- [x] Web Dashboard: Next.js, port 3333
- [x] File bridge: throttled 500ms Idling event
- [x] Shared WPF library: wpf_helper.py

## Planned (v1.1)

- [ ] Auto-dimensioning in sections and elevations
- [ ] NBS Specification linker — link wall/floor types to NBS clauses
- [ ] COBie readiness report — parameter completeness for handover
- [ ] Drawing issue manager — revision clouds, status stamps, issue registers
- [ ] IFC export validator — check model against IFC4 LOD before export
- [ ] Rhino.Inside integration guide — Grasshopper parametric geometry into Revit
- [ ] Clash detection summary — basic spatial checks across disciplines

## Planned (v2.0)

- [ ] Real-time dashboard with WebSocket (no manual refresh)
- [ ] Multi-model support (links, worksets)
- [ ] Family creation from description (not just placement)
- [ ] Rendered view generation via Enscape/V-Ray API hooks
- [ ] Parameter push/pull from external databases (COBie, Airtable)
- [ ] Voice input integration
